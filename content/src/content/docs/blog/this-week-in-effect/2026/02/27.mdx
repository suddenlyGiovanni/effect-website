## Technology

### Effect v4 Beta Updates

The v4 beta has been evolving rapidly since launch â€” here are the most notable changes that landed this week in effect-smol.

- **[New `Tx` modules & transaction model refactored](https://github.com/Effect-TS/effect-smol/pull/1515)**, improving consistency and ergonomics across transactional workflows.
- **Schema improvements**: added `Schema.makeOption` / `SchemaParser.makeOption `for constructing schema values as `Option`, introduced `OptionFromUndefinedOr` and `OptionFromNullishOr`, and fixed `Schema.encodeKeys` to correctly encode non-remapped struct fields.
- **OpenAPI & HTTP fixes**: multipart file upload schema fixed in OpenAI and OpenAI JSON schemas correctly transformed for AI integrations.
- **CLI bug fix**: resolved global flag handling in mixed CLI contexts.

You can follow the full changelog directly in the [effect-smol repository](https://github.com/Effect-TS/effect-smol).
